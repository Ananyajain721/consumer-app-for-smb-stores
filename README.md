# Consumer Application for SMB Kirana
SMB Consumer application aims to build the mobile web application for the NBU users which allows them easily connect with the grocery stores nearby. The application will enable the user to get the best price for order. This also provide the functionality to track the status of a placed order.
*This is an experimental project for STEP interns*
